FILE

 Clicking on this element in the menu bar, will appear a dropdown menu which proposes 3 choices:
  • New file: to open a new blank file in a new tab, with an ephemeral deafult name (until you save it).
  • Open file: to choose and open an existing file on your computer and upload it.
  • Save: to save on your computer the file you are working on.
  • Exitto quit the program.
