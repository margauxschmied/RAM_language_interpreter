SHORTCUTS

 To improve the use of our program we added 5 shortcuts to speed up the use of some commands.
  • Ctrl+Enter: executes the line in which there is the insertion cursor.
  • Ctrl-x: to execute the file.
  • Ctrl+S: to save the current file.
  • Escape: to quit the application.
  • Right-click on a line: allows us to either run the line, Mark that line as end of execution or Unmark the marked line.
